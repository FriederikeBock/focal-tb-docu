# Welcome to the FoCal TB tutorial

Welcome to the FoCal TB tutorial

This page is meant to make you familiar with the setup of the FoCal test beams executed during the last years, prepare you for upcoming test beams in terms of setting up the HGCROC readout as well as allowing you to simulate our test beam geometry using the central ALICE-software stack.
