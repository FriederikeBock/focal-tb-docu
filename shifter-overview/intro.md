## QUICKSTART: Run Checklist
1. Create new environment in [Control GUI](http://alio2-flp-focal.cern.ch:8080/)
2. Start run from within configured environment
3. Record run settings in the [google sheet](https://docs.google.com/spreadsheets/d/1CEVsBG8wgO2F9DYG3_Rz1IMYbPuPbzVct7_hPUwUYgk/edit?usp=sharing) and add them as a log in the [ALICE bookkeeping system](http://alio2-flp-focal.cern.ch:4000/)
4. In the [ALICE bookkeeping system](http://alio2-flp-focal.cern.ch:4000/), tag run with appropriate beam type and energy
5. Monitor [run QC](http://alio2-flp-focal.cern.ch:8082/) and [error log](http://alio2-flp-focal.cern.ch:8081/?q={%22message%22:{%22exclude%22:%22TRG%20error%20call=%5C%22RunList%5C%22%25%0A%0ASubTimeFrame%20deserialization%20failed.%25%22},%22severity%22:{%22in%22:%22E%20F%22}})
6. Log any issues during the run in the [ALICE bookkeeping system](http://alio2-flp-focal.cern.ch:4000/)
7. Stop run when needed from [Control GUI](http://alio2-flp-focal.cern.ch:8080/)
8. Shutdown environment (**do not kill!**)
